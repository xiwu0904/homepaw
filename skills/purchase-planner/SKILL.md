# 食材采购 (Purchase Planner)

根据 fridge-check 输出的缺少食材清单，在天猫超市搜索食材，选择新鲜品质好的，加入购物车，然后通知用户**自己去结算**。

## ⛔ 红线：绝对不要提交订单

**只加入购物车，绝对不要点击"结算"、"提交订单"、"去结算"、"立即购买"、"立即下单"等任何会产生实际付款的按钮。**
**看到任何包含"结算"、"下单"、"购买"、"付款"字样的按钮，一律不点。**
**你的任务到"加入购物车"为止，剩下的由用户自己完成。**

**必须用 browser 工具（profile: xhs），不要用 web_search。**

## 天猫超市注意事项

1. 天猫/淘宝经常有弹窗（广告、优惠券、登录提示），遇到弹窗时：
   - 先 snapshot 看清弹窗内容
   - 找到关闭按钮（通常是 X、关闭、我知道了、跳过）点击关闭
   - 如果找不到关闭按钮，尝试点击弹窗外的空白区域
   - 如果还关不掉，用 browser evaluate 执行：`document.querySelector('.baxia-dialog-close, .next-dialog-close, .close-btn, [class*="close"]')?.click()`
2. 天猫超市 URL: `https://chaoshi.tmall.com`
3. 搜索框通常在页面顶部
4. 加入购物车按钮通常是"加入购物车"或"加购"

## Step 1：获取缺少食材清单

从 fridge-check 的输出中获取 missing_items 列表。如果用户直接给了食材清单，直接使用。

## Step 2：打开天猫超市

```
browser 工具（profile: xhs）打开 https://chaoshi.tmall.com
```

打开后 snapshot 检查页面状态：
- 如果有弹窗 → 关闭弹窗
- 如果需要登录 → 提醒用户在 Chrome 中登录淘宝/天猫

## Step 3：逐个搜索食材并加入购物车

对每个缺少的食材，执行以下循环：

### 3a. 搜索食材

在天猫超市搜索框中搜索食材名称：
1. snapshot 找到搜索框的 ref
2. 点击搜索框，输入食材名（如"五花肉"）
3. 点击搜索按钮或按回车
4. 等待搜索结果加载

**搜索关键词策略：**
- 蔬菜/水果："{食材名} "
- 肉类："{食材名} "
- 调味料：直接搜食材名
- 蛋类："{食材名}"

### 3b. 选择商品

1. snapshot 获取搜索结果
2. 选择排名靠前、评价好、价格合理的商品（优先选择天猫超市自营）
3. 点击商品进入详情页

### 3c. 加入购物车

1. snapshot 检查商品详情页
2. 如果有弹窗 → 关闭
3. 如果需要选择规格（重量、数量等）→ 选择合适的规格
4. 找到"加入购物车"按钮并点击
5. snapshot 确认是否加购成功（通常会有提示）

### 3d. 返回继续下一个食材

加购成功后，返回天猫超市首页或搜索页，继续搜索下一个食材。

## Step 4：确认购物车（只看不点结算）

所有食材加购完成后：
1. 打开购物车页面：`https://cart.taobao.com/cart.htm`
2. snapshot 截取购物车内容
3. browser screenshot 截图购物车
4. **不要点击任何结算/下单按钮**

## Step 5：通知用户自行结算

向用户发送采购结果：
```
🛒 已将以下食材加入天猫超市购物车：

1. 五花肉 500g - ¥XX
2. 葱 1把 - ¥XX
3. 冰糖 1袋 - ¥XX

预计总价：¥XX

⚠️ 请你自己打开淘宝 APP 或天猫超市确认并结算，我不会帮你下单。
购物车链接：https://cart.taobao.com/cart.htm
```

**再次强调：任务到这里结束。不要帮用户结算、下单、付款。**

## 降级策略

- 天猫超市搜不到某食材 → 换关键词重试（去掉"新鲜"等修饰词）
- 加购失败（需要登录）→ 提醒用户登录后重试
- 商品缺货 → 记录并告知用户，建议线下购买
- 弹窗关不掉 → screenshot 截图给用户看，请用户手动处理后继续
